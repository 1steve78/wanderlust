# Airbnb Clone

### Techs Used:
1. Express.js (+ejs)
2. MongoDB (mongoose)
3. Passport

### How to run the app:
On your terminal, run the following commands:
1. `git clone https://github.com/1steve78/wanderlust.git`
2. `cd wanderlust`
3. `npm install`
4. `npm run dev`
5. On your browser, open http://localhost:8080
